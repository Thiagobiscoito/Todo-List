### INSTALAR O NODEMON

### INSTALAR O EXPRESS

### INSTALAR O DRIVE DO MONGO DB
npm install mongodb --save

### INSTALAR O CORS 
CORS é um pacote node.js para fornecer um middleware Connect / Express que pode ser usado para habilitar CORS com várias opções.

npm install cors
